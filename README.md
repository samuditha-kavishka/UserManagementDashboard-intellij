# User Management Dashboard

A simple **User Management Dashboard** built with Java and HTML/CSS in IntelliJ IDEA. This application allows you to add, edit, delete, and view users in a clean and responsive interface.


---

## Features

- List all users in a table
- Add a new user (name, email, role)
- Edit existing user details
- Delete a user
- View detailed user profile
- Responsive HTML/CSS interface
